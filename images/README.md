# Diagram Images

This directory contains rendered diagram images for documentation purposes.

## Viewing Draw.io Diagrams

All architecture diagrams are provided in draw.io format (`.drawio` files) located in the [diagrams](../architecture/diagrams/) directory.

### Online Viewing

1. Go to [draw.io](https://app.diagrams.net/) (also known as diagrams.net)
2. Click "Open Existing Diagram"
3. Select the `.drawio` file from the diagrams directory
4. View and edit the diagram as needed

### Local Viewing

1. Install [draw.io Desktop](https://github.com/jgraph/drawio-desktop/releases)
2. Open the `.drawio` file
3. View and edit the diagram

### Exporting Images

From draw.io:

1. Open the diagram
2. Go to File → Export as → PNG (or SVG, PDF)
3. Save the exported image

## Diagram Files

The following diagrams are available:

- `architecture.drawio` - High-level system architecture
- `component.drawio` - Detailed component relationships
- `deployment.drawio` - Production deployment topology
- `workflow.drawio` - Key user workflows
- `data-flow.drawio` - Data movement flows
- `sequence.drawio` - Authentication and pairing sequence

## Notes

- Draw.io files are XML-based and can be edited with any text editor
- Diagrams can be embedded in documentation using draw.io links
- For best results, use draw.io desktop or web application
